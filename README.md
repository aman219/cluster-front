# cluster-front
## A web based NAS fontend in reactjs. This is for my cluster project.

# Features
- Responsive UI
- Coustom audio player
- Coustom video player
- Image, Audio and video file supported.

### To run the application on local machine.

```
npm install
```
```
npm start
```
